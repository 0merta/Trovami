# Trovami

Live location app created in Flutter using Firebase as the backend 


##Features
    
    Create groups and choose to share your live location
    View group members on the map 
    Google Sign-in
    
##Preview




##Dependencies
    
    Flutter
    Firebase
    Google Sign in
    Maps Plugin

1.Setup Flutter
2.Clone the repo


3.Run the app
